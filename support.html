DataScrub — Installation

First-time setup

  1. Download the DataScrub64-packed.xll file
  2. Double-click the file to load it in Excel
  3. You should see a DATA SCRUB tab in the Excel ribbon

Make it load automatically every time

  1. In Excel, go to File > Options > Add-ins
  2. At the bottom, set "Manage" to Excel Add-ins and click Go
  3. Click Browse and select the DataScrub .xll file
  4. Click OK. DataScrub will now load every time you open Excel.

DataScrub — License Activation

How to activate

  1. Click the License button on the DATA SCRUB ribbon tab
  2. Enter your license key (format: DS-XXXXX-XXXXX-XXXXX)
  3. Click Activate. Your license is now active on this device.

Switching devices

Each license works on one device. To move it to a new computer:

  1. On your old computer: click License on the ribbon, then click Deactivate
  2. On your new computer: install DataScrub and enter the same license key

Common Issues

Ribbon tab not showing in Excel

  • Make sure you're using Excel 2016 or later on Windows (64-bit)
  • Try double-clicking the .xll file again
  • Check File > Options > Add-ins to see if it's listed as "Disabled". If so, enable it.
  • If you have both 32-bit and 64-bit Excel installed, make sure you're using the matching .xll file

License key not working

  • Make sure you're connected to the internet (required for first activation)
  • DataScrub keys start with DS-, NonprofitReports keys start with NPR-
  • If your subscription has expired, you'll need to renew at getdatascrub.com

"License already activated on another device"

Each license key is bound to one device. To switch:

  1. Open Excel on the old computer
  2. Go to the product's ribbon tab and click License
  3. Click Deactivate to free the device slot
  4. Now activate on your new computer
