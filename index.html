Frequently Asked Questions

Is there a free trial?

Yes! Both DataScrub and NonprofitReports include a free 3-day trial with all features. No credit card needed.

What versions of Excel do they work with?

Both products work with Microsoft Excel 2016, 2019, 2021, and Microsoft 365 on Windows (64-bit).

Is my data safe?

Absolutely. Both tools run 100% locally inside Excel. Your data never leaves your computer. No uploads, no cloud processing.

Can I use it on multiple computers?

Each license key activates on one device. If you switch computers, deactivate from the old one and reactivate on the new one. For teams, purchase multiple licenses.

How do I install it?

Download the .xll file, double-click it to load in Excel, then go to File > Options > Add-ins > Excel Add-ins > Browse and select the file to make it load automatically.

What happens when my subscription expires?

You'll be prompted to renew. Your data is never affected — the add-ins only modify cells when you tell them to.

What accounting systems does NonprofitReports support?

NonprofitReports auto-detects imports from QuickBooks Online & Desktop, Xero, Blackbaud Financial Edge, Sage Intacct, and Aplos. You can also import from any system via CSV or Excel export.

Do you offer refunds?

Yes. 14-day money-back guarantee on both products. No questions asked.
